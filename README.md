A console-based **Music Player Simulator** developed in **C**, created to practice classic **Data Structures** such as **Doubly Linked Lists**, **Stacks**, and **Queues**.

The application simulates the behavior of a music player without reproducing audio files, focusing on algorithms, dynamic memory allocation, modular programming, and pointer manipulation.

---

# ✨ Features

* 🎵 Music Library
* 📂 Playlist Management
* ⏯ Play Music
* ⏸ Pause / Resume
* ⏭ Next Song
* ⏮ Previous Song
* 🔀 Shuffle Mode
* 🔎 Search by Title
* 📜 Playback History
* 📋 Playback Queue

---

# 🏗 Data Structures

| Feature          | Structure          |
| ---------------- | ------------------ |
| Music Library    | Doubly Linked List |
| Playlist         | Doubly Linked List |
| Playback History | Stack (LIFO)       |
| Playback Queue   | Queue (FIFO)       |

Windows

